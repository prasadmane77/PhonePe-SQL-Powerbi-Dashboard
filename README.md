# PhonePe-SQL-Power BI-Dashboard
End-to-end data analysis project using  SQL for querying and Power BI  for inertactive dashboard

Download Power BI (.pbix ) file :
https://drive.google.com/file/d/1ND8s5ydNLN_0vWe8AW8LZV2wH6yB56NY/view?usp=drive_link

## Overview 
Multi-page Power BI dashboard analyzing digital transactions by service, month, status, and 
failure reason. 
## KPIs
- Successful transactions ~95–96%
- Failed transactions ~4–5%
- Total amount varies by category (50M to 3,333M)
- Monthly values show clear ups and downs  
## Key Insights 
- July shows highest transaction spike in multiple charts.
- February / August show lower transaction values.
- Loans category contributes the highest amount.
- Recharge & Bills is the lowest contributing service.
- Failed reasons are mainly Server Error, Wrong PIN, and Insufficient Amount.
- Trends are not flat → user activity changes month to month.
